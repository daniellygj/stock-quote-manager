# Atock quote manager

### API documentation:
After running the project, access the url http://localhost:8080/swagger-ui.html


### How to run the project:
 * To run on docker:
    ```
      mvn install
    
      docker-compose up -d
    ```
    
  * To run on local machine:
    ```
      mvn install
    
      mvn spring-boot:run
    ```
